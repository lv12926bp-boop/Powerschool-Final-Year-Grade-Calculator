# Powerschool-Final-Year-Grade-Calculator
Made in CMU CS Academy. Project for AP Computer Science Principles.
To use this program you must copy and paste the code into CMU CS Academy.
When you run the program you will be prompted to switch to whatever class you want to calculate your grade for.
After you have chosen your desired class, input all 5 grade for each quarter and final exam.
At the end of all input your final year score will be visible.
All done with 100% accuracy, based on original Powerschool calculating formulas.
NOTE: This code has only been run and tested on CMU CS Academy. You may attempt to run the code on a different platfom (must be Python language!). ONLY AT YOUR OWN RISK.
